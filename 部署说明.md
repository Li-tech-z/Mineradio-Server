# Mineradio 服务器部署指南（Render 免费版）

## 前置要求
- GitHub 账号（免费）
- Render 账号（免费，用 GitHub 登录）

## 部署步骤

### 1. 上传到 GitHub
```bash
cd D:\code\Mineradio-Server
git init
git add .
git commit -m "Mineradio server"
git remote add origin https://github.com/你的用户名/Mineradio-Server.git
git push -u origin main
```

### 2. 部署到 Render
1. 打开 https://render.com
2. 用 GitHub 账号登录
3. 点击 "New +" → "Web Service"
4. 选择你的 Mineradio-Server 仓库
5. 设置：
   - Name: `mineradio`
   - Runtime: `Node`
   - Build Command: `npm install`
   - Start Command: `node server.js`
6. 点击 "Create Web Service"
7. 等待部署完成（约2-3分钟）

### 3. 获取服务器地址
部署成功后，Render 会给你一个地址，格式如：
`https://mineradio-xxxx.onrender.com`

### 4. 在手机 APK 中设置
1. 打开 Mineradio APK
2. 点击右上角设置图标
3. 输入服务器地址：`https://mineradio-xxxx.onrender.com`
4. 保存

## 注意事项
- 免费版有 15 分钟无活动后休眠的限制，首次访问会慢几秒
- Cookie 登录状态会在服务器重启后丢失（需重新登录）
- 每月 750 小时免费额度，够正常使用
